<h1 align="center">REAL-TIME ETL</h1>

  <p align="center">
    Real-time ETL pipeline collecting international flight data through NiFi, streaming it via Kafka to Python for processing, then forwarding it via another Kafka stream to Elasticsearch for storage. Finally, custom dashboards in Kibana enable real-time monitoring of the entire dataset.

The entire project is containerized using Docker services for seamless deployment and execution across diverse environments.
